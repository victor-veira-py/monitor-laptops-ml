# 🚀 Monitor de Laptops Dell Latitude - Mercado Libre

Este sistema automatizado detecta ofertas de laptops en tiempo real, evaluando especificaciones técnicas frente al precio para encontrar las mejores oportunidades.

## ✨ Características Principales
- **Web Scraping Dinámico**: Desarrollado con **Selenium** para extracción de datos en tiempo real.
- **Manejo de Sesiones (Cookies)**: Sistema de persistencia para optimizar el acceso y evitar bloqueos de seguridad.
- **Algoritmo de Valor "UFC"**: Lógica personalizada que rankea los productos según RAM, CPU y estado técnico.
- **Alertas en Tiempo Real**: Integración con **Telegram API** para notificaciones instantáneas al celular.
- **Reportes Profesionales**: Generación de archivos Excel organizados con **Pandas** y **XlsxWriter**.

## 🛠️ Tecnologías Usadas
- Python 🐍
- Selenium WebDriver
- Telegram Bot API
- Pandas & XlsxWriter

## ⚙️ Configuración
Para ejecutar este bot, sustituye los valores de `TOKEN_TELEGRAM` y `CHAT_ID` en el script principal por tus propias credenciales.
